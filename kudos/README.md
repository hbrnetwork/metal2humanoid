# Metal2Humanoid Kudos System

Welcome to the Metal2Humanoid contributor reward system.

This folder tracks **open tasks** that earn community points ("kudos").

Contributors accumulate points by completing meaningful tasks posted here.

Points may unlock:

- Public recognition in the main repo  
- Tier-based swag  
- Invitation to contributor calls  

---

# How It Works

1. Choose a task from the table below.
2. Reach out to the listed Verifier/Contact.
3. Submit a PR referencing the task.
4. Once approved, points are awarded.
5. Your name is added to the Contributor Board.

---

# Contribution Tiers

| Tier | Points | Recognition |
|------|--------|------------|
| Builder | 10+ | Name listed in repo |
| Architect | 100+ | Highlighted contributor |
| Core Contributor | 1000+ | Contributor call invite |
| Founding Builder | 10000+ | Special recognition |

Points accumulate over time.

# Proposing New Tasks

If you believe something important is missing:

1. Open an issue titled: `KUDOS PROPOSAL: <task name>`
2. Describe scope and expected deliverables.
3. Suggest a point value.
4. Maintainers will approve and add it to the table.

---

# Rules

- Points are awarded only after merge.
- Partial credit may be granted for substantial effort.
- Maintainers reserve the right to adjust point values.
- All contributors are rewarded full points.
